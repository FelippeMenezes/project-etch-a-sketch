# project-etch-a-sketch
A simple sketchpad app inspired by Etch-a-Sketch, built to practice DOM manipulation, JavaScript logic, and interactive UI. Part of The Odin Project journey.

## 🚀 Live Demo
[View the Live Preview here](https://felippemenezes.github.io/project-etch-a-sketch/)

## 🛠️ Technologies Used
*   **HTML5:** Basic structure of the application.
*   **CSS3:** Grid layout and styling.
*   **JavaScript:** DOM manipulation, event handling, and dynamic grid creation.
*   **Git:** Version control and commit history.
*   **GitHub:** Repository hosting.

## 📝 Features
*   Interactive grid that changes color on hover.
*   Dynamic grid size based on user input.
*   Clear/reset functionality.
*   Responsive and simple UI.

## 🧠 What I Learned
During this project, I reinforced my knowledge of:
*   Manipulating the DOM using JavaScript.
*   Handling events like mouseover and clicks.
*   Dynamically creating and modifying elements.
*   Structuring interactive front-end logic.

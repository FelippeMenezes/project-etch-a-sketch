# project-etch-a-sketch
A simple sketchpad app inspired by Etch-a-Sketch, built to practice DOM manipulation, JavaScript logic, and interactive UI. Part of The Odin Project journey.

## 🚀 Live Demo
[View the Live Preview here](https://felippemenezes.github.io/project-etch-a-sketch/)

## 🛠️ Technologies Used
*   **HTML5:** Basic structure of the application.
*   **CSS3:** Grid layout and styling.
*   **JavaScript:** DOM manipulation, event handling, and dynamic grid creation.
*   **Git:** Version control and commit history.
*   **GitHub:** Repository hosting.

## 📝 Features
*   **Interactive Drawing:** Grid squares change color on hover to simulate a real sketchpad.
*   **Rainbow Mode:** Generates a random RGB color for each square, creating vibrant and unique artwork.
*   **Progressive Shading:** Each interaction adds 10% black/opacity, allowing for depth and sketching effects.
*   **Eraser Tool:** Allows the user to clear specific cells and refine the drawing.
*   **Dynamic Grid Scaling:** Users can set the grid resolution (up to 100x100) to balance detail and performance.
*   **Clear/Reset Canvas:** A dedicated button to wipe the entire grid and start fresh.

## 🕹️ How to Use
1.  **Initialize:** Open the app and enter the desired grid size when prompted (e.g., 16 for 16x16).
2.  **Draw:** Move your cursor over the canvas to start sketching.

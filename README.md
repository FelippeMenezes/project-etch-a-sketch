# project-etch-a-sketch
A simple sketchpad app inspired by Etch-a-Sketch, built to practice DOM manipulation, JavaScript logic, and interactive UI. Part of The Odin Project journey.

## 🚀 Live Demo
[View the Live Preview here](https://felippemenezes.github.io/project-etch-a-sketch/)

🛠️ Technologies Used
HTML5: Structure and layout of the application.
CSS3: Styling and grid layout.
JavaScript: DOM manipulation, event listeners, and dynamic grid generation.
Git: Version control and commit history.
GitHub: Repository hosting.
📝 Features
Interactive grid that responds to mouse hover.
Dynamic grid resizing (user-defined size).
Color-changing drawing effect.
Clear/reset button to restart the sketch.
🧠 What I Learned

During this project, I reinforced my knowledge of:

Manipulating the DOM with JavaScript.
Handling user events like mouseover and clicks.
Creating elements dynamically with JavaScript.
Managing application state in a simple UI.
Writing cleaner and more organized JavaScript code.
